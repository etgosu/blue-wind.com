계정 삭제 방법

사용자는 앱 내 설정 메뉴에서 언제든지 계정 삭제를 요청할 수 있습니다.

삭제 절차:

  1. 앱 실행
  2. 카메라 화면 우측 상단 설정(⚙️) 아이콘 선택
  3. 하단의 "회원 탈퇴" 버튼 선택
  4. "탈퇴" 텍스트 입력 후 확인 버튼 클릭

계정 삭제 시 Google 계정 연동 정보 및 동의 이력이 즉시 영구 삭제됩니다. 삭제된 데이터는 복구되지 않습니다.

앱 보관함에 저장된 사진 및 동영상은 기기 내 앱 전용 저장공간에 보관되며, 앱을 삭제하면 함께 삭제됩니다. 중요한 파일은 계정 삭제 전 갤러리로 내보내기 하시기 바랍니다.

사진·동영상 개별 삭제

사용자는 계정을 삭제하지 않고도 보관함에 저장된 사진 및 동영상을 개별적으로 삭제할 수 있습니다.

  1. 앱 실행
  2. 카메라 화면에서 보관함 버튼 선택
  3. 삭제할 사진 또는 동영상 선택 (다중 선택 가능)
  4. 하단의 "일괄 삭제" 선택
  5. 확인 버튼 클릭

선택한 파일은 기기에서 즉시 영구 삭제되며 복구되지 않습니다. 삭제 전 갤러리로 내보내기를 권장합니다.
