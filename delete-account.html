계정 삭제 방법

사용자는 앱 내 설정 메뉴에서 언제든지 계정 삭제를 요청할 수 있습니다.

삭제 절차:

  1. 앱 실행
  2. 설정 메뉴 이동
  3. 계정 삭제 선택
  4. 확인 버튼 클릭

계정 삭제 시 모든 체온 및 투약 기록 데이터는 즉시 영구 삭제됩니다. 삭제된 데이터는 복구되지 않습니다.

기록 개별 삭제

사용자는 계정을 삭제하지 않고도 체온 및 투약 기록을 개별적으로 삭제할 수 있습니다.

  1. 앱 실행
  2. 타임라인 화면 이동
  3. 삭제할 기록 선택
  4. 하단 메뉴에서 "삭제" 선택
  5. 확인 버튼 클릭

선택한 기록은 즉시 영구 삭제되며 복구되지 않습니다.
