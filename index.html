내 옷장 + 오늘 날씨로
AI가 “지금 입기 좋은 코디”를 추천합니다.

Blue Wind는 옷장 관리와 날씨 기반 코디 추천을 한 번에 해결하는 앱을 만들고 있어요. 사용자가 등록한 옷(소재·두께·색상 등)과 날씨를 바탕으로, AI가 맥락 있는 코디를 제안하고 필요하면 이미지 합성/생성으로 “착장 느낌”까지 빠르게 확인할 수 있게 합니다.

제품 살펴보기 기술 스택
핵심 가치 “내 옷을 기준으로” 오늘의 선택을 쉽게
타깃 옷이 많은데 매일 뭐 입을지 고민하는 사용자
차별점 날씨+옷장+AI 추천+시각화(이미지)
원칙 개인정보 최소 수집 · 안전한 저장

제품: 옷장 기반 코디 추천

MVP에 가까운 핵심 기능부터 단단하게 만들고 있어요.

WEATHER

날씨 기반 추천

기온·강수·체감 요소를 반영해 오늘 “현실적으로 입을 수 있는” 코디를 제안합니다.

WARDROBE

내 옷장 관리

상의/하의/신발을 등록하고 소재·두께·색상·통기성 같은 특성을 기반으로 추천 정확도를 올립니다.

AI

개인화 코디

사용자 취향을 반영해 스타일 힌트·상황(출근/주말/데이트)까지 고려한 조합을 제공합니다.

IMAGE

시각화(이미지)

코디를 입었을 때의 느낌을 빠르게 이해할 수 있도록 이미지 합성/생성 옵션을 준비합니다.

COST

비용 최적화

API 사용량/모델 선택/캐시 설계로 이미지 생성 비용과 지연을 줄이는 전략을 적용합니다.

UX

반응형 UX

모바일 중심으로 빠르고 단순한 흐름: “옷 선택 → 추천 → 저장/공유”까지 자연스럽게.

기술 스택

빠른 배포와 안전한 데이터 처리를 우선합니다.

CLIENT

Front

  • 모바일 앱 중심의 일관된 UX
  • API 호출/캐시/상태관리로 반응성 확보
BACKEND

Functions

  • 서버리스 운영
  • 정책(RLS)과 토큰 기반 인증으로 접근 제어
AI

LLM / 이미지 모델

  • 텍스트 기반 추천(코디/요약/퀴즈 등)
  • 이미지 생성/합성 기능은 비용·품질 균형을 최적화
OPS

운영

  • 배포/버전 관리
  • 관찰 가능성: 호출량/에러/지연 모니터링

* 이 페이지는 Home Pages(정적)로 운영 중이며, 앱/백엔드는 별도로 구성됩니다.

로드맵

가볍게 시작해서, 유저 피드백으로 빠르게 개선합니다.

Phase 1 · MVP

옷장 등록 → 날씨 기반 추천 → 추천 저장/히스토리 → 기본 UI/UX 안정화

Phase 2 · 개인화

취향/상황 반영, 코디 조합 다양화, 필터/정렬 고도화, 추천 설명(왜 이 조합인지) 강화

Phase 3 · 시각화

이미지 합성/생성 기능을 “옵션”으로 제공하고, 비용 최적화(캐시/모델/빈도 제어) 적용

보안·프라이버시

개인 데이터는 최소 수집 + 안전한 저장을 기본으로 합니다.

PRIVACY

최소 수집 원칙

  • 추천 품질에 필요한 데이터만 수집
  • 민감정보는 기본적으로 수집하지 않음
SECURITY

안전한 접근 제어

  • 토큰 기반 인증 + RLS 정책으로 사용자별 데이터 분리
  • 비밀키/관리자 키는 클라이언트에 포함하지 않음
