내 옷장 + 오늘 날씨로
AI가 "지금 입기 좋은 코디"를 추천합니다.

Blue Wind는 옷장 관리와 날씨 기반 코디 추천을 한 번에 해결하는 패션 앱입니다. 사용자가 등록한 옷(소재·두께·색상 등)과 오늘의 날씨를 바탕으로, AI가 맥락 있는 코디를 제안하고 이미지로 착장 느낌까지 빠르게 확인할 수 있습니다.

기능 살펴보기 사용 흐름 보기 문의하기
핵심 가치 "내 옷을 기준으로" 오늘의 선택을 쉽게
타깃 사용자 옷이 많은데 매일 뭐 입을지 고민하는 분

주요 기능

MVP의 핵심 기능부터 단단하게 만들고 있습니다.

WEATHER

날씨 기반 추천

기온·강수·체감 요소를 반영해 오늘 "현실적으로 입을 수 있는" 코디를 제안합니다.

WARDROBE

내 옷장 관리

상의·하의·신발을 등록하고 소재·두께·색상·통기성 같은 특성을 기반으로 추천 정확도를 높입니다.

AI

개인화 코디

취향과 상황(출근·주말·데이트 등)을 반영해 맥락 있는 코디 조합을 제안합니다.

IMAGE

코디 시각화

코디를 입었을 때의 느낌을 이미지 합성·생성으로 빠르게 확인할 수 있습니다.

HISTORY

추천 히스토리

지난 코디 추천을 저장·기록해 자신의 스타일 패턴을 파악하고 활용할 수 있습니다.

UX

직관적인 인터페이스

모바일 중심으로 빠른 흐름: "옷 선택 → 추천 확인 → 저장/공유"까지 자연스럽게.

사용 흐름

4단계로 오늘의 코디를 완성하세요.

1

옷 등록

갖고 있는 옷을 소재·색상·두께별로 등록해 내 옷장을 구성합니다.

2

날씨 확인

앱이 현재 위치의 날씨를 자동으로 가져와 오늘의 기온·강수 등을 분석합니다.

3

AI 코디 추천

날씨와 내 옷장 데이터를 바탕으로 AI가 상황에 맞는 코디 조합을 제안합니다.

4

확인 & 저장

이미지로 착장을 확인하고 마음에 드는 코디를 저장하거나 공유합니다.

로드맵

가볍게 시작해 유저 피드백으로 빠르게 개선합니다.

Phase 1 · MVP

옷장 등록 → 날씨 기반 코디 추천 → 추천 저장·히스토리 → 기본 UI/UX 안정화

Phase 2 · 개인화

취향·상황 반영, 코디 조합 다양화, 필터·정렬 고도화, "왜 이 조합인지" 설명 강화

Phase 3 · 시각화

이미지 합성·생성 기능 제공, 비용 최적화(캐시·모델·빈도 제어) 적용, 코디 공유 기능

보안 · 프라이버시

개인 데이터는 최소 수집 + 안전한 저장을 기본으로 합니다.

PRIVACY

최소 수집 원칙

  • 추천 품질에 필요한 데이터만 수집
  • 민감정보는 기본적으로 수집하지 않음
  • 사용자 데이터 삭제 요청 즉시 처리
SECURITY

안전한 접근 제어

  • 토큰 기반 인증 + RLS 정책으로 사용자별 데이터 분리
  • 비밀키·관리자 키는 클라이언트에 포함하지 않음
  • 전송 구간 암호화(HTTPS) 적용
